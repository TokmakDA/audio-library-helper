#1.0.2
- Добавлена Firefox-совместимая схема запуска без отдельного манифеста: теперь проект использует единый `manifest.json` для Chromium и Firefox.
- Background-процесс адаптирован под оба окружения: Chrome запускает `service_worker`, Firefox использует `background.scripts`.
- Убраны основные Firefox-блокеры загрузки: ошибка `importScripts is not defined`, потеря runtime-соединения и предупреждения от устаревшего Firefox-манифеста.
- Обновлены иконки действия и документация по загрузке временного расширения в Firefox.

#1.0.1
- Восстановлена рабочая логика расширения на базе исходного проекта с сохранением текущей стилистики, тем и вынесенного конфига.
- Скачивание и получение данных о треках снова переведены в service worker, чтобы убрать проблемы с CORS и нестабильным page context.
- Popup переработан: `header / main / footer`, условный рендер состояний без template-зависимости, единый стиль именования id и селекторов.
- Метаданные popup теперь берутся из manifest, а базовые тексты страницы настроек живут в общем конфиге.
- Улучшена читаемость карточки текущего трека: контраст текста, кнопки и подсказок теперь подстраивается под цвет трека.

#1.0.0
- Первый публичный релиз Audio Library Helper.
- Проект переименован и приведён к единому названию в манифестах, интерфейсе и документации.
- Обновлена иконка расширения.
- Интерфейс popup и страницы настроек приведён к актуальному виду.
