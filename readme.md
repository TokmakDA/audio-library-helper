# Audio Library Helper

Расширение для экспорта аудиотреков в личную библиотеку.

[![Скачать ZIP](https://img.shields.io/badge/Скачать-ZIP-f0ad4e?style=for-the-badge)](../../archive/refs/heads/main.zip)

## Установка в Chromium-браузеры

1. Откройте страницу расширений:
   `chrome://extensions/`, `opera://extensions/` или `edge://extensions/`.
2. Включите режим разработчика.
3. Нажмите `Загрузить распакованное расширение`.
4. Выберите папку проекта с файлом `manifest.json`.

## Установка в Firefox

1. Откройте `about:debugging#/runtime/this-firefox`.
2. Нажмите `Загрузить временное дополнение...`.
3. Выберите `manifest.json`.

Firefox загружает это расширение как временное дополнение до перезапуска браузера.

## Структура

- `manifest.json` — общий манифест для Chromium и Firefox
- `js/` — логика расширения
- `html/` — popup и страница настроек
- `icons/` — иконки расширения

## Документация к коду

Основная логика проекта задокументирована через JSDoc-комментарии в файлах `js/*.js`, чтобы было проще поддерживать код, разбирать поток сообщений между скриптами и вносить изменения без поиска по всему проекту.
