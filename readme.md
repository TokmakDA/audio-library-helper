# Audio Library Helper

Расширение для скачивания треков из Яндекс Музыки в личную библиотеку.

Работает только с Яндекс Музыкой.

[![Скачать ZIP](https://img.shields.io/badge/Скачать-ZIP-f0ad4e?style=for-the-badge)](../../archive/refs/heads/main.zip)

## Что умеет

- Добавляет кнопки `Скачать` рядом с треками на странице Яндекс Музыки.
- Показывает в popup текущий трек и соседние треки: предыдущий и следующий, если сервис уже подготовил их для интерфейса.
- Позволяет скачать текущий, предыдущий или следующий трек, а также альбом, плейлист, дискографию артиста или чарт.
- Поддерживает горячую клавишу `Shift + D` и скачивание по двойному клику для текущего трека.
- Умеет экстренно остановить большую очередь загрузки прямо из popup.

## Установка

Ниже описана установка локальной распакованной версии расширения.

Перед установкой:

1. Скачайте проект архивом или клонируйте репозиторий.
2. Распакуйте архив в отдельную папку.
3. Убедитесь, что внутри этой папки лежит файл `manifest.json`.

### Google Chrome

1. Откройте страницу расширений: введите в адресной строке `chrome://extensions`.
2. Включите `Режим разработчика`.
3. Обычно переключатель находится справа вверху страницы.
4. Нажмите `Загрузить распакованное расширение`.
5. Выберите папку проекта, в которой лежит `manifest.json`.

### Microsoft Edge

1. Откройте страницу расширений: `edge://extensions/`.
2. Включите `Developer mode`.
3. Переключатель обычно находится справа вверху.
4. Нажмите `Load unpacked`.
5. Выберите папку проекта с `manifest.json`.

### Opera

1. Откройте страницу расширений: `opera://extensions`.
2. Включите `Режим разработчика`.
3. Обычно переключатель расположен справа вверху.
4. Нажмите `Загрузить распакованное расширение`.
5. Выберите папку проекта с `manifest.json`.

### Yandex Browser

1. Откройте страницу расширений: `browser://extensions/`.
2. Включите `Режим разработчика`.
3. Обычно переключатель находится справа вверху.
4. Нажмите `Загрузить распакованное расширение`.
5. Выберите папку проекта с `manifest.json`.

### Firefox

1. Откройте `about:debugging#/runtime/this-firefox`.
2. Нажмите `Load Temporary Add-on`.
3. Выберите файл `manifest.json` внутри папки проекта.

Важно: в Firefox это временная установка. После перезапуска браузера расширение нужно загрузить заново.

## Первый запуск

1. Откройте Яндекс Музыку.
2. Если вкладка Яндекс Музыки уже была открыта до установки расширения, обновите её.
3. Нажмите на значок расширения.
4. Если popup попросит авторизацию, подтвердите её один раз.
5. Включите любой трек, чтобы popup получил данные о текущем воспроизведении.

## Как пользоваться

### Скачать один трек на странице

1. Откройте Яндекс Музыку.
2. Найдите нужный трек.
3. Нажмите кнопку `Скачать` рядом с ним.

### Скачать текущий трек

Есть три способа:

1. Нажать `Скачать` в карточке `Текущий трек` в popup.
2. Нажать `Shift + D`.
3. Сделать двойной клик по странице Яндекс Музыки вне кнопок и полей ввода.

### Скачать предыдущий или следующий трек

1. Откройте popup расширения.
2. Под карточкой `Текущий трек` отображаются компактные блоки `Предыдущий трек` и `Следующий трек`, если данные доступны.
3. Нажмите `Скачать` напротив нужного трека.

### Скачать альбом, плейлист, артиста или чарт

1. Перейдите на нужную страницу в Яндекс Музыке.
2. Откройте popup расширения.
3. В правой колонке появится контекст страницы:
   альбом, плейлист, артист или чарт.
4. Нажмите кнопку скачивания.
5. Для плейлистов можно ограничить диапазон треков прямо в popup.

### Экстренно остановить большую очередь

1. Откройте popup во время загрузки.
2. Нажмите `Экстренная остановка`.
3. Расширение перестанет запускать новые загрузки и попытается отменить уже начатые.

Примечание: если часть файлов уже передана в менеджер загрузок браузера, некоторые треки могут успеть завершиться до полной остановки.

## Рекомендации для массового скачивания

- Отключите в браузере запрос папки или имени файла перед каждым скачиванием. Иначе при скачивании альбома или плейлиста браузер будет постоянно открывать всплывающее окно.
- Лучше заранее выбрать отдельную папку для музыкальной библиотеки в настройках браузера или системы.
- Для длинных очередей удобнее сначала открыть нужный альбом или плейлист, а потом запускать пакетное скачивание из popup.
- Если хотите быстро остановить очередь, держите popup под рукой: кнопка остановки работает прямо из окна расширения.

## Если что-то не работает

- Если popup не видит Яндекс Музыку, откройте сервис и обновите вкладку.
- Если не показывается текущий трек, запустите воспроизведение и откройте popup ещё раз.
- Если после установки кнопки `Скачать` не появились, обновите страницу Яндекс Музыки.
- В Firefox временное расширение исчезает после перезапуска браузера — это нормальное поведение.
